# Amazon_Sales_Dashboard_Project
📊 Amazon Sales Analysis Dashboard (Power BI)
Project Overview
This project presents an interactive sales analysis dashboard built using Power BI based on an Amazon sales dataset.
The dashboard helps analyze sales performance, profit trends, product categories, shipping modes, and regional performance across different countries and markets.

It enables businesses to quickly understand where sales are coming from, which regions perform best, and how shipping methods impact sales.

# 📊 Sales Analysis

# 🌍 Market Insights

Regional sales performance


# 🚚 Shipping Analysis

# 💸 Discount Analysis
Discount distribution by country

# 🎛 Interactivity

Interactive charts and drill-down analysis
